SterPaint

SterPaint nyitóoldal

A SterPaint egy sztereogram-készítő és kísérleti festőprogram. A program mellett itt közvetlenül elérhetők a minta Pattern és DepthMAP fájlok is.

A SterPaint program jobb felső sarkában egy kis Samples gomb található. Erre kattintva bármikor visszatérhetsz erre a nyitóoldalra, ahonnan a Pattern és DepthMAP minták böngészhetők.

Az alábbi galériák a GitHub repo aktuális tartalmából épülnek fel, ezért a mappák frissítése után automatikusan az új fájlok jelennek meg.

SterPaint opening page

SterPaint is a stereogram generator and experimental painting tool. In addition to the program itself, sample Pattern and DepthMAP files are available here directly.

A small Samples button appears in the upper-right corner of the SterPaint program. Clicking it returns to this page, where the Pattern and DepthMAP sample galleries can be browsed.

The galleries below are built from the current GitHub repository contents, so newly uploaded files appear automatically.

SterPaint.html Galéria / Gallery Pattern DepthMAP

Röviden

  • A program a SterPaint.html linken indítható.
  • A képek és minták az alábbi automatikus galériából tölthetők le.
  • A GitHub Pages nem listáz mappát közvetlenül, ezért kell ez a galériaoldal.
  • A repo-ba új fájlok feltöltése után pár perc kellhet, mire minden frissül.

In brief

  • Start the program from the SterPaint.html link.
  • Sample files can be downloaded from the automatic galleries below.
  • GitHub Pages does not expose folder listings directly, so this gallery page is needed.
  • After uploading new files, GitHub Pages may need a few minutes to refresh.
