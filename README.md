# SterPaint-beta GPT-SHAME

Minimal GitHub Pages keret a SterPaint futtatásához.

## Tartalom
- `SterPaint.html` – a futtatható egyfájlos app
- `DepthMAP/` – ide teheted a betölthető mélységképeket (PNG/BMP/PIC stb.)
- `Pattern/` – ide teheted a betölthető mintákat

## GitHub Pages
1) Repo Settings → Pages
2) Source: **Deploy from a branch**
3) Branch: `main` / Folder: `/ (root)`
4) Mentés után a site URL a Pages résznél megjelenik.

Tipp: a GitHub Pages alapból `index.html`-t keres. Ha szeretnéd, hogy rögtön induljon:
- nevezd át a `SterPaint.html`-t `index.html`-re, VAGY
- tartsd meg így, és nyisd: `/SterPaint.html`
